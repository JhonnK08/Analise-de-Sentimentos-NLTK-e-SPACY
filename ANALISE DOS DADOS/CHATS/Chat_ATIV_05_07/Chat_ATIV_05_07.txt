Tem-se 1 - a = 95%, então a = 5% e a / 2 = 2,5%. O coeficiente de confiança que deve ser buscado na distribuição t com ? = n - 1 = 25 - 1 = 24. Esta é a linha da tabela. A coluna poderá ser o valor a = 5% ou então o valor a / 2 = 2,5%, dependendo do tipo de tabela. Em qualquer caso o que se procura é o valor “t” com grau de liberdade igual a 24, isto é, o valor t24 tal que:
P(- ta/2 < t24 < ta/2) = 95% Este valor vale 2,064. (Note-se que na a normal este mesmo valor valia 1,96). Então o intervalo de confiança de 95% para a média desta população será:
[X - ta/2 S n ; X + ta/2 S n ] = [50 - 2,064.10/5; 50 + 2,064.10/5] = [50 - 4,13; 50 + 4,13] = [45,87; 54,13], ou seja, pode-se afirmar com uma certeza de 95% de que este intervalo conterá a média desta população.

Uma população tem um desvio padrão igual a 10 e média desconhecida. Uma amostra de tamanho n = 100 é retirada e fornece uma média x = 50. Qual o intervalo de 95% de confiança para a média desta população? Solução: Têm-se 1 - a = 95%, então a = 5% e a / 2 = 2,5%. O coeficiente de confiança que deve ser buscado na normal padrão é valor za/2 de Z tal que: P(Z > za/2) = 2,5%, ou então: F(-za/2) = 2,5% ? -za/2 = F -1(2,5%) = -1,96. Então o intervalo de confiança de 95% para a média desta população será: [X - za/2 s n ; X + za/2 s n ] = [50 - 1,96.10/10; 50 + 1,96.10/10] = [50 - 1,96; 50 + 1,96] = = [48,04; 51,96], ou seja, pode-se afirmar com uma certeza de 95% de que este intervalo conterá a média desta população. Obs.: O valor e = za/2 s n  é denominado de erro padrão da estimação. Não confundir com o valor s n  que é o erro padrão da amostragem. O erro padrão da estimação é a semi-amplitude do intervalo de confiança. A amplitude do intervalo de confiança (IC) será; 2e.
